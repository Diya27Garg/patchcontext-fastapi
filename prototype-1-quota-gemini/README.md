\# Prototype 1 — Quota-Based (Gemini API)



Status: Deprecated — see \[`prototype-2-local`](../prototype-2-local) for the current, final version.



This was the first working version of PatchContext, using the Gemini API end-to-end

for both embeddings and generation. It was superseded due to API quota limits that

made indexing and evaluating the full FastAPI commit/PR/issue history impractical.

See the main \[README](../README.md) for the full pivot story.
